Glossary

  HTML Attributes
Contain information about an HTML element beyond the content.
  HTML Head
Contains non-displayed information about the document.
  Favicons
Used to visually distinguish your site in tabs and bookmarks.
  HTML Comments
Chunks of text within HTML code that is ignored by web browsers.
  Document Fragments
Specified parts of an HTML document can be referred to as document fragments.
